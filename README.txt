===========================================================================================================================

Requirements: 
-Beautifulsoup4
-Requests
-Operator
-Discord
(All are installable with pip install)

===========================================================================================================================

Bot Command(s):
- !help to display command list
- !newegg [item name(include space)] [budget, if no budget enter 0.0]
- !amazon [item name(include space)] [budget, if no budget enter 0.0]


===========================================================================================================================

We have learned the following:
-Utilize beautifulsoup and request to scrape information from websites
-Understand and use operator to extract data from the dictionary where scraped data is stored
-Use and apply the Discord API to provide an interactive bot as a platform for the program
